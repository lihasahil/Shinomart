# ShinoMart

Simple Prject based on Next js to fetch product, search, filter

## Live Demo

Public Deployment: [<https://your-deployed-link-here.com>](https://shinomart.vercel.app/)


## Run the Project Locally

### 1. Clone

    git clone https://github.com/your-username/your-repo.git](https://github.com/lihasahil/Shinomart

### 2. Enter folder

    cd shinomart

### 3. Install dependencies

    npm install

### 4. Start dev server

    npm run dev

App runs at http://localhost:3000

##  Build for Production

    npm run build
